Churn Prediction¶

Discription¶

Churner is generally defined as a customer who stops using a product or service for a given period of time. The definition of churner will vary on the type of the organization. Like in case of Game Industry churner will be one who has not login to their server within a particular interval or the one who has not move to next level in the game. In case of flipkart service churner will be one who has not visited the website within particular interval or the one who has not ordered anything on it in a given interval of time. This interval will vary based on the organization and based on the application the user is involved in.

The predictive model should be robust enough to handle the variance of interval of not using a serive or product very intelligently.

Aim : Do the analysis of the given dataset and build the model to investigate and predict which of the customers are more likely to leave the bank soon using different Machine Learning algorithms and compare their results.

2. Handling categorical data¶

From the above analysis we have seen that the feature Surname might contribute significant role in predicting churners. It has 2932 unique elements out of 10000 which means most of them are repeated only. But inorder to use this in any machine learning algorithm we need convert in numerical form because machine will not understand the text, it understands only number. So by using Label encoding convert it into numbers.
