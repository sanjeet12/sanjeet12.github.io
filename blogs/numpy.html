It works efficiently with arrays and metrices in python.

5. Generating points using linspace and arange¶

np.arange(start,stop,step,dtype=None)
It returns an array of evenly spaced values. stop is excluded.

np.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None)
The linspace function is similar to arnage function of numpy except that we will tell it how many numbers we want to return and it will split up the equally space between the given interval.

There are num equally spaced samples in the closed interval [start, stop] or the half-open interval [start, stop) (depending on whether endpoint is True or False)

9. Stacking Array and creating column vector¶

Stacking of a array can be done in two forms
a. vertical stacking
Stack arrays in sequence vertically (row wise).
This is equivalent to concatenation along the first axis after 1-D arrays of shape (N,) have been reshaped to (1,N). Rebuilds arrays divided by vsplit.
b. horizontal stacking
Stack arrays in sequence horizontally (column wise).
This is equivalent to concatenation along the second axis, except for 1-D arrays where it concatenates along the first axis. Rebuilds arrays divided by hsplit.
